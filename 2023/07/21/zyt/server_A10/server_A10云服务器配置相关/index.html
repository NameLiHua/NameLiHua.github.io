mysql可能出现的错误

[ERR] 1273 - Unknown collation: ‘utf8mb4_0900_ai_ci’
大致原因：sql对应的mysql版本高于当前要导入的mysql库的版本，引发的1273错误。

解决方案：

打开sql文件，将文件中的所有utf8mb4_0900_ai_ci替换为utf8_general_ci，同时将utf8mb4替换为utf8。处理sql后重新在mysql库中导入sql文件即可。
